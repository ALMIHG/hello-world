# Кефирный Роботик

Более игровая версия детской браузерной игры для возраста **5–10 лет**.

## Что добавлено
- Главный герой: яркий дружелюбный **кефирный роботик** с большими глазами и «булькающим» кефирчиком внутри.
- Прогрессия по возрастам: блоки от **5 до 10 лет**, в каждом по **10 уровней**.
- На уровне нужно:
  1. Собрать **3 монетки** на сцене,
  2. Правильно решить задачу,
  3. Дойти до портала справа и нажать кнопку завершения уровня.
- Управление:
  - `←` и `→` — движение
  - `↑` — прыжок
  - `Q` — подсказка
  - `W` — повтор задания
  - `E` — радостный танец роботика
- Добавлены экран старта и on-screen кнопки управления (удобно для планшета).

## Как запустить (самый простой способ)
1. Откройте файл `start.html`.
2. Нажмите большую кнопку **«▶ Запустить игру»**.

## Альтернатива: через локальный сервер
```bash
python3 -m http.server 8000
```
После запуска откройте:
- `http://127.0.0.1:8000/start.html` — страница с кнопкой запуска
- `http://127.0.0.1:8000/index.html` — сама игра
# КЕФИРЧИК

Детская браузерная игра для возраста **5–10 лет**.

## Что внутри
- Главный герой: дружелюбный роботик **КЕФИРЧИК** с яркими эмоциями.
- Игровой цикл: роботик движется к звёздочкам, а вопрос появляется у каждой звезды.
- При правильном ответе: звезда засчитывается, КЕФИРЧИК радуется, играет короткая мелодия.
- При неверном ответе: КЕФИРЧИК грустит и визуально «сдувается».
- Дополнительно: препятствия, прыжки и бонусы по пути.

## Управление
### Клавиатура
- `←` и `→` — движение
- `↑` — прыжок
- `Q` — подсказка (когда открыт вопрос)

### Экранные кнопки (смартфон и ПК)
- `⬅️` — движение влево (удержание)
- `➡️` — движение вправо (удержание)
- `⬆️` — прыжок
- `💡 Подсказка` — показать подсказку у активного вопроса

## Запуск
Откройте `index.html` в браузере.

## Локальная smoke-проверка
Перед отправкой в PR рекомендуется запускать:

```bash
bash scripts/smoke-check.sh
```

Проверка включает:
- синтаксический контроль JavaScript из `index.html` (`node --check`),
- проверку наличия ключевых элементов управления в HTML.
