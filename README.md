# Кефирный Роботик

Простая детская браузерная игра для возраста **5–10 лет**.

## Что внутри
- Главный герой: яркий дружелюбный **кефирный роботик** с большими глазами и «булькающим» кефирчиком внутри.
- Прогрессия по возрастам: блоки от **5 до 10 лет**.
- В каждом возрасте — **10 уровней** с простыми заданиями по возрасту.
- Простое управление:
  - `←` и `→` — движение
  - `↑` — прыжок
  - `Q` — подсказка
  - `W` — повтор задания
  - `E` — радостный танец роботика

## Запуск
Откройте файл `index.html` в браузере.
